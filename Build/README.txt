All required folder must be present for this to work.

1-Folder containing all the .walk files named 'Walks'.
	This is used as an input for the Walk2Vec step and an output from the RDF2Walk step.

2-Folder containing all the .vec files named 'Vectors'.
	This is used as an input for the Vec2Cluster step and an output from the Walk2Vec step.

3-Folder containing all the .cluster files named 'Clusters'.
	This is used as an output from the Vec2Cluster step.
